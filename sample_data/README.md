# Sample data (demo mode)

Everything in this directory is **synthetic**. It exists so the dashboard
can run with no API keys, no network access, and no rate-limit risk —
useful for public demos, GitHub reviewers, CI, and screenshots.

- `prices/<TICKER>.csv` — synthetic OHLCV series generated by
  `generate.py` (deterministic seeds).
- `prices/BDRY.csv` — synthetic series used as the BDRY (BDI proxy)
  fixture.
- `fundamentals.json` — fictitious per-ticker fundamentals.
- `news.json` — fictitious headlines written for this project. No
  copyrighted text.
- `watchlist.json` — ticker → company map.

To regenerate: `python3 sample_data/generate.py` (run from the repo root).

> **None of these values represent real market or news data.** The
> dashboard surfaces a clear "DEMO" badge whenever this fixture set is in
> use.
